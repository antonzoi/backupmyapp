Backupmyapp
===========

Requirements:

	net/ssh, net/sftp

Plugin for backup service.

	script/plguin install git://github.com/workisfun/backupmyapp.git
	script/generate backupmyapp KEY=your key from service
	
Then, test connection with server:

	rake backupmyapp:test
	
Enjoy using it!
